# 📅 Interactive Calendar Application

A responsive and interactive calendar component built using React and Vite. This project focuses on clean UI, smooth user interaction, and efficient state management.

## 🚀 Features

* 📆 Monthly calendar view
* 🔄 Navigate between months
* 📝 Add and manage notes/events
* 🎯 Highlight current date
* 📱 Fully responsive design
* ⚡ Fast performance with Vite

## 🛠️ Tech Stack

* React (Functional Components + Hooks)
* Vite
* JavaScript (ES6+)
* CSS / Tailwind (if you used it, keep it — else remove)

## 📂 Project Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/calendar-app.git
cd calendar-app
npm install
```

## ▶️ Running the App

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173/
```

## 📦 Build for Production

```bash
npm run build
```
